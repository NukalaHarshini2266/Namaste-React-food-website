# 🍔 Food Ordering App (Namaste React Project)

**Namaste React Learning Journey**

This project is a **React-based food ordering application** built while learning from the **Namaste React course by <a href="https://github.com/akshaymarch7">Akshay Saini</a>, Founder of NamasteDev.**.  
It includes core React concepts, modern frontend practices, and testing using Jest.

A big thank you to **Akshay Saini sir** for the amazing explanations and practical approach to learning React.

---

## 🚀 Live Demo
🔗 Deployed Link: https://namastefoodweb.netlify.app/

---

## 📌 Features

- Display list of restaurants
- Search restaurants by name
- Filter Top Rated Restaurants
- Show Promoted Restaurants using HOC
- Restaurant Menu Page
- Online / Offline status detection
- User Context for managing user data
- Redux for Cart management
- Add items to cart
- Dynamic routing using React Router
- Lazy loading (code splitting)
- Shimmer UI for loading state
- Unit testing using Jest and React Testing Library
